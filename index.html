Project Basic Browser JavaScript

This project is a web application using 100% JavaScript. This is a game program made with the enjoyment of creating donuts on a click! Player uses button clicks to create donuts. The donuts can be used as currency to buy auto clickers and donut multipliers, which will be used to make even more donuts!

Project Studio Inventory Node.js and C#

This project was designed and implemented by a team of six students, including myself. It is a single page web application separating the content seen by regular users vs. administrators. Project started out as in .NET MVC but for the flexibility, we created the frontend structured in Node.js and backend structured in C#. The app, Studio Access Ally, is an inventory management system made for any recording studio.

Project Album Collection Node.js and C#

Working in a pair programming format utilizing the Agile (Scrum) methodology, this web application was designed using RESFful API within the ASP.Net Entity Framework. The frontend is a single page web application created in Node.js. User can interact with the album collection database to create, view, edit and delete artists, songs and albums.
